# SHANGSHIELD.ML :dragon: :camel:

![GitHub CI](https://github.com/alyxshang/shangshield.ml/actions/workflows/ocaml.yml/badge.svg)

***An OCaml package to measure the security of passwords. :dragon: :camel:***

## ABOUT :books:

This repository contains an OCaml package to check the security of your passwords using the Shang algorithm.

## INSTALLATION :inbox_tray:

To use ***Shangshield.ml*** in your OCaml project, please execute the following steps in order and make sure you have Opam, Dune, and the OCaml compiler installed:

- 1.) Clone this repository.
- 2.) Change directory into the repository's root.
- 3.) Run the command `opam install .` to install ***Shangshield.ml*** into your Opam switch.
- 4.) To use the package in your code, please add the word `shangshield` to your `dune-project` file and any `dune` files inside different modules.
- 5.) The package's namespace can be imported with `open Shangshield`.

## USAGE :hammer:

To view this package's API, you can read the main source file, [`lib/shangshield.ml`](lib/shangshield.ml), located in the `lib` directory.
